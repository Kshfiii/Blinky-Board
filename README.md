# Blinky-Board
A Heart-shaped Blinky Board with 16 LEDs
# Components:
  1x 555 Timer  
  16x LEDs  
  x1 10 μF Capacitor   
  x1 100k Variable Resistor   
  x1 10k Resistor   
  x1 330 ohm resistor    
  x1 Push Button    
  x1 Battery
